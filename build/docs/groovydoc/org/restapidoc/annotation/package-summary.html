Package org.restapidoc.annotation

  • Annotation Type Summary 
    Annotation Type Description
    RestApi This annotation is to be used on your "service" class, for example controller classes in Spring MVC.
    RestApiBodyObject This annotation is to be used on your method and represents the parameter passed in the body of the requests
    RestApiError This annotation is to be used inside an annotation of type ApiErrors
    RestApiErrors This annotation is to be used on your method and represents the possible errors returned by the method
    RestApiHeader This annotation is to be used inside an annotation of type ApiHeaders
    RestApiHeaders This annotation is to be used on your method and contains an array of ApiHeade
    RestApiMethod Override ApiMethod to hbe allowed to have have no value for some fields
    RestApiObject This annotation is to be used on your object classes and represents an object used for communication between clients and server
    RestApiObjectField
    RestApiObjectFields
    RestApiParam This annotation is to be used inside an annotation of type ApiParams
    RestApiParams This annotation is to be used on your method and contains an array of ApiParam
    RestApiResponseObject This annotation is to be used on your method and represents the returned value
